# Audio for Minna Lesson 1

The page currently uses the browser Web Speech API and does not require audio files.

To replace a vocabulary item with a local recording:

1. Record or license a Japanese pronunciation file.
2. Save it in this directory, for example `watashi.mp3`.
3. Add `audioSrc: 'assets/audio/minna/lesson-01/watashi.mp3'` to the vocabulary item.
4. Keep the source, author, and license information beside the files.

Suitable sources:

- Original recordings made for VieJap.
- Lingua Libre / Wikimedia Commons recordings with compatible Creative Commons attribution.
- Materials explicitly licensed for redistribution.

Do not upload textbook audio or third-party MP3 files when redistribution permission is unclear.
